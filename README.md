# data-automation-portfolio
This repository contains small tools I've built to automate data processing and analysis tasks for personal use, including JSON-based roster manipulation and language learning vocabulary data extraction with filtering using a known words list.

### Vocabulary Mining Tool

This tool processes structured language data to extract, organize, and analyze vocabulary for learning purposes. It ingests input data (e.g., text or subtitle-derived content), parses relevant terms, and outputs structured datasets that can be used for review, frequency analysis, or further processing.

The project demonstrates practical experience with:
- Data parsing and transformation
- Working with structured and semi-structured data
- Automating repetitive data extraction workflows
- Organizing outputs for downstream analysis

This tool was built as part of a broader effort to support language learning through data-driven methods.
